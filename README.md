# AVR-Assembly
The source code used to demonstrate a rudimentary assembly implementation of a I2C driver to communicate with a LCD Display.
build.sh just calls some of the binaries in the AVR GCC Toolchain to compile and send an executable to the ATMEGA328P chip over USB

Link to the full article [here](https://open.substack.com/pub/volenmironov/p/full-metal-programming?r=7efg46&utm_campaign=post&utm_medium=web&showWelcomeOnShare=true).
